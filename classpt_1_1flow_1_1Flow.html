Detailed Description

A single processing unit in the flow framework. It's connected to other Flow objects like a linked list, but can have more than 1 flow object next in line.

Data gets consumed by it, processed and changed (type, value or both) Then it gets produced to the next Flow objects in line.

The produced data gets spread to the next nodes in different ways based on the given ProductionPolicy pt::flow::ProductionPolicy
