std::shared_ptr< F > pt::flow::Pipeline::add ( std::shared_ptr< F >  f)
inline

Add a flow object to the pipeline

Pipeline::add will auto-wire all sources to the first module / aggregator.
Pipeline::add will auto-wire all modules and aggregators linearly.
Pipeline::add will auto-wire the last module / aggregator to all sinks.

Usage:

  1. Add ALL sources.
  2. Add ALL modules / aggregators.
  3. Add ALL sinks.
Parameters
fpt::flow::Flow object to add to the pipeline
Returns
The same flow object that was added to the pipeline
Exceptions
exceptions::unordered_sourcesWhen a source is added after a module/aggregator/sink.
exceptions::unknown_flow_objectWhen an unknown object derived from Flow is added.
